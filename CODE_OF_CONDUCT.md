# Code of Conduct

This project expects respectful, professional collaboration.

## Expected behavior

- Be direct and constructive.
- Focus criticism on ideas, code, and execution.
- Assume good intent, but do not lower the technical bar.
- Respect the security-sensitive nature of the project.

## Unacceptable behavior

- Harassment, abuse, or personal attacks
- Bad-faith disruption
- Publishing secrets, private keys, or sensitive project data

## Enforcement

Project maintainers may edit, hide, lock, or remove content that violates these expectations.
