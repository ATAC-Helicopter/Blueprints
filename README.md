# Blueprints

Blueprints is a local-first, version-centric release planning desktop application for developers and small teams.

## Stack

- .NET 8
- Avalonia UI
- MVVM

## Current State

The repository currently includes:

- product and implementation planning documents
- solution and project scaffold
- starter domain, storage, security, and collaboration contracts
- initial Avalonia desktop shell

## Projects

- `Blueprints.App`
- `Blueprints.Core`
- `Blueprints.Storage`
- `Blueprints.Security`
- `Blueprints.Collaboration`
- `Blueprints.Tests`
